# Digital Competency of Adult Educators in Kenya

##  Overview
This project analyzes survey data from **1,162 adult educators** across Kenya to assess digital competency, infrastructure gaps, and policy needs.  
The study provides a **baseline dataset** for future interventions and includes **visualizations, dashboards, and recommendations**.

##  Tools & Technologies
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- Power BI / Tableau
- Excel
- KoboToolbox (data collection)

##  Key Findings
- 59.4% of educators rated their digital skills as *basic*; only 11.9% advanced.
- 71.6% relied on personal devices (mostly smartphones).
- 77% had never attended government digital training.
- 91.6% recommended revising adult education policy to include digital literacy.

##  Deliverables
- **Python notebooks**: Data cleaning, descriptive stats, cross-tabulations.
- **Dashboards**: Interactive Power BI visuals for policymakers.
- **Report**: Evidence-based recommendations for training and policy.
